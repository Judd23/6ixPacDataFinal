# Survey codebooks

This folder stores the official HERI survey codebooks used for the STAT 520 analyses.

- `2000FreshmenSurvey_Codebook.pdf` — Variable definitions and response coding for the entering freshmen survey.
- `2004SeniorSurvey_Codebook.pdf` — Codebook for the senior follow-up survey administered to the same cohort.

Use these references when interpreting column names in the data files or when updating the variable table workbook.
