This repository contains the full workflow for the 6ix Pac Data Dawgs, STAT 520 project, including data management, exploratory analysis, factor analysis, structural modeling, machine learning, visualizations, and the final report materials. The goal of our repository design is to make the project accessible for us new learners, while supporting advanced reproducible statistical analysis.

Folder focus: `docs/`

Central reference materials live here: codebooks, survey instruments, SAS program summaries, and any narrative documentation that supports the analysis.

Key contents - (Among others)
- `Codebooks/` for official HERI variable definitions and encoding.
- `Survey_Instruments/` for the questionnaires administered to each cohort.
- `sas_output/` and other reference notes that explain the lineage of the raw data.

Usage guidelines
- Treat this directory as read-only
- When new documentation arrives, place it in the appropriate subfolder
